Reads data from a specified register of the Gyro device.

This function communicates with the device over SPI, sending the register address and then receiving the data from the device. The SPI read operation is executed by setting the RW bit to {\ttfamily 1} as per the datasheet. The CS pin is used to select the device, and SPI transactions are done with the SPI handle.
